## Get Started

App require [Node.js](https://nodejs.org/)

Install dependencies

```sh
npm i
npm start
```

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:8080](http://localhost:8080) to view it in the browser.
